## Overview

This template supports the preparation of a Maturity Level 3 (ML3) compliance report against the [ACSC Essential Eight](https://www.cyber.gov.au/resources-business-and-government/essential-cyber-security/essential-eight). It is technology-agnostic and suitable for any organisation seeking to document full alignment with ACSC intent.

ML3 represents comprehensive, automated, real-time security controls fully aligned with ACSC guidance. Reports at this level document not only compliance status but also automation maturity, real-time detection and response capability, continuous improvement programs, and threat intelligence integration.
