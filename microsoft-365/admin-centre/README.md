# Microsoft 365 Admin Centre

Documentation for the Microsoft 365 admin centre (admin.microsoft.com) — the primary portal for tenant-level administration including licensing, domain management, organisational settings, service health monitoring, and user management.

---

## Planned Content

- Licensing management and assignment workflows
- Domain configuration and DNS requirements
- Organisational settings and tenant-wide defaults
- Service health monitoring and incident response
- User and group management (admin centre perspective — Entra ID specifics belong in `identity/`)
- Billing and subscription management
- Support ticket workflows

---

## Boundary

This folder covers **admin.microsoft.com** functionality only. For related topics:

- Entra ID user/group management → `identity/`
- Security & Compliance centre → `security/` or `compliance/`
- Workload-specific admin centres (Exchange, SharePoint, Teams) → sibling folders in `microsoft-365/`
