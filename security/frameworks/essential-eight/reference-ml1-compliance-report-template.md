## How to Use This Template

This template provides a structured reporting format for documenting an organisation's compliance with the Australian Cyber Security Centre (ACSC) Essential Eight at Maturity Level 1 (ML1). It is technology-agnostic — the structure applies regardless of the tooling, platform, or assessment method used by the organisation.

Complete each section by replacing placeholder text (shown in `[Square Brackets]`) with assessed values. Guidance notes in *italic* describe the intent of each field.

Status values used throughout:

| Symbol | Meaning |
| ------ | ------- |
| Compliant | Requirement fully met |
| Partial | Requirement partially met — gaps exist |
| Non-Compliant | Requirement not met |

### Appendix B: Compliance Scoring Methodology

**Scoring Criteria:**

- **Compliant (100%):** All requirement criteria are met with documented evidence. No gaps identified.
- **Partial (50–99%):** Some requirement criteria are met. Identified gaps do not fully satisfy the requirement.
- **Non-Compliant (0–49%):** Requirement criteria are not met or insufficient evidence exists to confirm compliance.

**Control Weighting:**

- All eight controls are weighted equally at ML1.
- Overall compliance is calculated as the average of all control compliance scores.
