## Boundary

This folder covers **OneDrive admin centre** functionality. For related topics:

- The OneDrive admin centre's "Mobile application management" settings configure a special Intune App Protection **Global policy**. These MAM settings are **deprecated** in the OneDrive admin centre — Microsoft recommends configuring them directly in Intune. Documentation for App Protection Policies belongs in `endpoints/intune/`, not here.
- OneDrive app configuration via Intune → `endpoints/intune/`
- Microsoft Graph OneDrive API → `integrations/`
